# 仙逆 · 结构化世界观知识库

本仓库把网络小说**《仙逆》**(耳根 著,全 2088 章)的世界观整理成**结构化数据**:将人物、分身、法宝、功法、神通、傀儡、势力、地点、星域、境界、纪元、事件等从原文中抽出,建成可查询、可溯源、可随剧情演化的**实体-关系图谱**。

---

## 目录结构

```
taxonomy.json                本体定义:类别 / 实体类型 / 关系类型
kb.json                      知识库元信息
docs/<类别>/<单元>/           每个抽取单元一目录
  ├─ knowledge.json            结构化内容:摘要、关键事实、涉及实体、章节范围
  └─ content.md                该单元的正文梳理
entities/<类型>/              实体档案,按类型分目录(每个实体一个 JSON)
graph/relations.jsonl        全部关系边(主语-关系-宾语 + 出处 + 章节序)
sources/<单元>/               每个单元对应的原文切片(供溯源核对)
log.md                       内容变更日志
```

## 本体设计要点(区别于一般修仙志)

《仙逆》基调冷硬,母题为**逆天 / 因果 / 轮回 / 虚幻**。本体在通用修仙体系上做了针对性扩展:

- **分身体系**:`avatarOf` 表达分身—本尊关系(王林招牌)。
- **身份延续**:`reincarnationOf`(轮回/寄魂/夺舍)、`impersonates`(冒名)分别建模。
- **因果与击杀**:`kills`(击杀链)、`devours`(吞噬修为)、`controls`(操控傀儡 puppet)、`inherits`(传承)。
- **地图与空间**:除包含关系外,增 `connectsTo`(传送阵/裂缝连通)、`borders`(接壤)、`leadsTo`(单向通往),并设 `passage`(通道)、`region`(星域)实体——支撑真实舆图绘制。
- **时间线**:`era`(纪元 太古/远古/上古)、`precedes`(先后)、`causes`(因果)、`duringEra`,区分叙事章序与故事时序(应对重度闪回)。
