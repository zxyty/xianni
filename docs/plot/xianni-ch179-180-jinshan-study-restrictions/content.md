# 仙逆 第179-180章 学习禁制 / 倒霉的古帝

## 背景：闯关第二关「禁山」
- 王林于第一关得**极品灵石**后，因情况危急、第一关难以再度侥幸通过，遂选择继续前行而非原路返回；推断此地应有返回传送阵（千年前孟驼子四人得以返回），但孟驼子玉简未载此地信息，传承物品不在孟驼子手中。
- 第二关名为**禁山**：整座山峰皆由禁制构成，越往上禁制越复杂、威力越大、可通行路线越少；山顶有巨大漩涡通往**第三关**。

## 禁制之理
- 禁制是阵法的一种，但更具灵活性、更倾向神念；大神通者神念一动即可布下，历千万年只要神念不灭仍运转，甚至施术者死后其神念可自主成新意识维持运转。
- 破除二法：①**强力破解**（需极强修为）；②**研究破解**（了解制作原理与规则）。王林用第二法——受益于其身为「香魂」时对基础阵法的钻研。

## 王林的钻研历程（约十年）
- 从山脚最简单的杂草禁制起，逐草逐纹记录于玉简、循序渐进；对首处「强力杀敌」禁制，从只能坚持三息到能安全存活十息并化解红光。
- 自创**残影之圈**破解术：右手数息内晃动上万次连成一圈，每圈本身即一个禁制；后发展为「心神推演」，无需肉眼、凭圈内神识波纹即可察知禁制结构；最多可同时画出**十个残影之圈**（依赖极境神识之妙）。
- 每破一处禁制必在其上加设数层，并封死禁制间缝隙，令后来者即便化神期也可能因连环触发而死；身后强力叠加禁制有三四处。
- 成果：记录禁制的玉简超过一百块；修为由**结丹期**不知不觉进至**结丹中期**（距元婴期更近一步）；神识增强，练就「**神识之眼**」（眼内包罗万象、偶有日月星辰，端木极若见必惊）；头发四年前始白、十年后全白；气质由「万载玄冰」再添「深不可测」。
- 用黑色毒剑破解「以灵兽（蜈蚣）为禁制」的手法，据此认出施术者为**六欲魔君**（回忆其在通道内以身边年轻人为受术者施展诡功）。

## 三位闯关者
- **王林**：结丹中期，禁制造诣同辈无人能比，兴趣从「度关」转为「学禁制之术」，视禁山为循序学习的绝佳宝地。
- **六欲魔君**：化神中期，六欲魔功已至化境（最不惧情绪波动，故不归路如儿戏）；五百年前得宝可施**水遁术**（效同汪清越的土行舟），第一关入冰川之地轻松度过；随身带一年轻人作第三关关键祭品；曾被禁山某禁制困五年方脱；一路亦额外添加禁制。
- **古帝**（本章「倒霉」主角）：千年前即化神中期，为当年闯关主力强者，视六欲魔君等为晚辈；当年逃出时身受重伤未能杀三人。第一关入火行之地（本性相克）耗七年但保住大部分法宝；于**修魔海**养伤千年、修为寸步未进（疑伤及根基），改头换面拜入数个禁制门派系统学禁制。目标：闯完四关入**古神体内**得**婴变丹**，一跃成**婴变期**强者。

## 古帝之「倒霉」
- 三年来古帝饱受山峰固有禁制 + 王林/六欲魔君额外恶毒禁制之苦：破解一个常引发连环开启、虚实真假迷乱心神、无路可退只能硬破；第一关省下的法宝在此耗去大半。
- 古帝两年前判定上方有两位禁制高手，把最忌惮者由六欲魔君改判为神秘人（即王林），并惊觉此人似「刚学禁制」却禁制之术突飞猛进、隐成「快、诡」体系。
- 天空禁制禁止飞行/瞬移（千年前一化神初期修士瞬移即被紫色闪电连元婴一并击灭），故众人只能步行而上。
- 王林停留近一年布置**隐匿气息**的禁制（非攻击型），等身后化神期之人先过；近距离观察古帝破禁制而有所悟。
- **章末**：古帝行至距王林十丈之外，目光一闪盯向王林所在，露出一丝古怪之色（尚未识破）。

## 关键设定点
- 极境「无所不用其极」，支撑王林达成十个残影之圈。
- 掌握禁制后可自行布置同类禁制，威力或稍逊但诡异不减，是一股强大力量。
