# 仙逆 第235章 因为，他姓藤

## 一、返乡祭亲（凡人故乡·山村）
- 雨夜，白发青年王林现身恒岳山脉玄道宗山下密林，远望玄道宗大殿后转身，前往数百里外的一处小山村——他四百年前的凡人故乡。
- 山村村舍多经数代翻修，物是人非；惟王家祖屋与当年一模一样（实为后人依原样重修）。
- 祖屋正厅供两座灵牌：上首『王天水、周英素之灵』（王林父母），下首『长子王林之灵』。
- 王林上三炷香、跪拜磕头，立誓：『下次来时，以藤家全族人头，为爹娘搭建藤家人头塔。』杀气弥漫正厅。

## 二、偶遇四叔后代（王天土一脉）
- 一辆马车载来一位病态柔美的王家小姐（李伯赶车、丫头翠儿相随）。祖训：每年今日嫡系子孙须亲来此地祭拜。
- 王林现身盘问。女子自称家父王云飞，祖屋灵牌之人为其先祖之兄全家；其先祖名**王天土**（王林四叔）。
- 四叔王天土：中年被**飘渺宗**一位仙长看中，下山后在京城落根，成皇族供奉之一，王家自此在京城发展；98岁去世。其子**王虎**在王天土死后第三年亦亡。
- 王林感念四叔（除父母外最挂念之人），认下自己为其先祖，履行『若修仙有成，守护子孙后代以报赐仙之恩』的旧约。

## 三、赐福护佑
- 王林挥手驱散女子体内纠缠二十载的至阴顽疾（其母怀她时被人打伤所致）。
- 赠丹药一批：共七十二粒，嫡系子孙此生限服一粒，该女可服三粒。
- 取女子一滴血，留一枚**玉简**：内封一丝极境神识烙印，可攻杀赵国境内任何修士（无人能在其攻击下活过一时三刻），限用三次，非王家嫡系血液无法开启，需谨慎使用。随后遁去。
- 女子服丹后病态尽去、面色红润；老者与翠儿察觉异常。女子心有猜测，欲回京查族谱佐证。

## 四、灭族复仇正式开启（赵国）
- 出村后王林杀机大盛。谋略：不直接杀藤化元（恐树倒猢狲散、后代四散，且无法泄恨），要让藤化元眼睁睁看子孙一个个死绝、饱受悲痛后再取其魂。
- 神识扫遍赵国，锁定藤化元所在的**藤家城**；在其方圆万里插下**十六杆阵旗**布阵，令藤家城『只能进，不能出』。
- **唤魂神通**：王林咬破指尖弹血，凝出远古魔神虚像，喝『藤厉之魂，现』，唤出被其所杀之藤厉的残魂（无记忆、仅本能，时限半炷香）。此为古神**涂司**记忆中的小神通（涂司原视其无用，仅用于暂持法宝之威）。
- 王林吞下藤厉之魂，借其血脉感应，神识锁定全赵国所有拥有藤家血统之人——无论嫡系旁系，乃至藤家女子外嫁所生之子，只要有一丝藤家血统皆现。所谓灭族＝从根部抹杀整条藤家血统。

## 五、天道门七杀
- 王林踩**蚊兽**杀向最近门派**天道门**（坐落奇峰、顶峰大殿），此处有藤家血脉七人。
  - **藤玄**：藤家第六代子孙，结丹初期，天道门一元婴期始祖的入门弟子（另有五名藤家弟子仅筑基期）；与其妹**藤幽**（合欢宗弟子）长期乱伦私通。
- 王林以**禁幡**震碎防御光幕，宣告『在下与藤家子孙私人恩怨，挡路者，死』，神识威压令众元婴修士口吐鲜血。
- 依次诛杀七名藤姓子孙：广场惊惧青年、殿内悲愤青年、丹房姿色女子、奇峰外黑衣青年、白发筑基后期大圆满老者（以黑色飞剑刺杀）、后山正欢好的藤玄与藤幽。每杀一人皆以**魂旗**收魂、**蛟龙筋**捆尸，七尸拖于蚊兽之后如孔雀血尾。
- 阻拦的红脸元婴始祖被王林极境神识震碎神识、连元婴碎成肉块；王林冷言『阻止者，与藤家同罪』，余者不敢近。
- 天道门元婴始祖惊惧不已，事后掌教下令：召回门下弟子，中断与藤家一切产业往来，与娶藤家女双修的弟子断绝关系、逐出门派，从此与藤家再无关联。众人叹『藤家要完了，赵国要变天了』。

## 六、下一步
- 王林目标坚定，行灭族不容妇人之念。下一处目标为数万里外、藤家子孙多达**九十三人**的门派。
- 意图：慢慢地杀，让藤化元困于丧亲之痛却无法外出救治的苦海，尝当年王林撕心裂肺之痛。
