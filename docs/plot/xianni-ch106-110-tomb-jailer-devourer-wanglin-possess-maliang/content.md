# 仙逆 第121-122章 剧情梳理

## 一、修真联盟与六级修真国体系（设定铺陈）
- **修真联盟**：庞大组织，成员国数目仅极少数人知晓。成为成员国最低标准=达到六级、拥有修真星。
- **管理层级**：六级以下修真国均隶属本星球的六级修真国管理（联盟无精力逐一管理）。每个五级国控制众多四级国，四级国控制更多三级国。一、二级修真国联盟明文规定不许干涉，任其发展。
- **朱雀国 / 朱雀星**：朱雀国用近十万年从一级升至六级，为耗时最少的六级修真国之一。晋六级后获赠一颗星球，被联盟命名为**朱雀星**。在朱雀星上朱雀国的声音即神灵意志。

## 二、域外战场制度
- 朱雀星共有**十八个五级修真国**。五级国达标一刻，六级国赠送**十个**域外战场 → 全星仅**一百八十个**。
- 每坍塌一个即**永久少一个**，除非花费庞大物资向六级国购买。
- 域外战场如何开辟、空间裂缝通向何处，五级国无人得知。
- 开辟域外战场的法术在晋升六级时由联盟统一赠送，**严禁泄露给六级以下国家**，违者除名。

## 三、赵国的从属链与征召
- **赵国**=朱雀星三级国家 → 受控于四级修真国**青龙国** → 青龙国背后掌控者=五级修真国**巨魔族**。
- 每次域外大战，三级国家所有**元婴期高手**被征召，不得拖延反抗。当年**恒岳派五位始祖**即被如此征召，去则恒岳派可保、五人生死未知，不去则门派被灭。几乎每个三级国家皆如此。

## 四、历史母题：上古神话时期的覆灭
- 上古神话时期修真讲**天道、道心**，追求飘逸出尘的'道'。一场突如其来的浩劫使其灭亡，**修真联盟蓦然崛起**，'道'被完全粉碎。
- 联盟证明无欲无求的修真注定灭亡，唯有**弱肉强食、物竞天择**方能生存；当下修真者实力与势力远超上古神话时期。

## 五、域外战场坍塌与吞魂现世
- 巨魔族某域外战场内异变：三年内获得神识的修士神识被抽走，如流星雨般向某方向冲去。
- 已处坍塌边缘的战场承受不住，裂开巨大口子，如被无形巨口吞噬。
- **三个庞大神识**从空间裂缝钻出——正是王林在空间裂缝内的三个'邻居'（即**吞魂**）。随之涌出大量**神秘物种（烟状生物/游魂）**，见人就扑、见物就上，如蝗虫；被扑中者魂飞魄散、身体萎缩成干尸。

## 六、王林神识归体与'不知死活'
- 王林三年分散的神识一道道回归，每回归一份神识壮大一分，渐达曾经程度（极少数因不在域外战场无法召唤）。
- 游魂扑向王林，王林睁眼冷喝**'不知死活'**，神识散出吞噬游魂；另一吞魂神识欲阻拦，王林不退反追，尽数吞噬。

## 七、与三吞魂的协议
- 王林神识一扫传出波动：'诸位，过界了'。三吞魂传意识：
  - 承认王林强大，惊讶他真从'那里'出来。
  - 阐明**吞魂**使命：诞生起唯一使命是吞噬坍塌空间，防止坍塌大范围蔓延；王林亦是吞魂之一。
  - 即便它们不吞，也会有其他吞魂来吞。
- 王林借**马良生前记忆**得知**传送阵开启将至**，若战场彻底坍塌恐传送阵无法开启。遂请求**放缓最终坍塌时间**。三吞魂同意（但游魂猎食不可阻止）。

## 八、夺舍马良之身
- 王林现身为**夺舍马良**而来。此身体天资灵根比王林之前好数倍，但需一段时间**祭炼**方可完全占据。
- 体内无半点灵力，全靠**神识**驱动。眼下要务：尽快找闭关之地，祭炼身体、提高修为，配合强大神识**杀回赵国**。

## 九、救下战神殿三人
- 王林循马良记忆飞向传送阵，途中游魂一律惊散不敢阻拦。
- 西北方向：**周紫虹**（三级修真国**火焚国·战神殿**弟子，一向恬静）与两名师兄（含**杨师兄**，约三十余岁）被十余游魂追杀，灵力枯竭、面露绝望。
- 战神殿原十人，除四天前独自离开的**马良**外，三天内只剩此三人。
- 王林以马良之貌出现（胸口衣襟有血污，眉清目秀带冷意），神识喝'滚'驱散全部游魂。周紫虹一度荒诞地猜'游魂惧怕马良'旋即否定，三人劫后余生松气。
- 伏笔：周紫虹回忆马良胆小怕死、脸皮厚，被彤师妹设法弄到域外战场，马良曾赠她一粒称可'驻颜'的丹药（真假未知）。

## 关键因果
- 域外战场坍塌 → 吞魂出世 → 王林神识归体 → 夺舍马良 → 请求延缓坍塌保传送阵 → 意图杀回赵国。

# 第122-123章 域外战场撤离与夺舍暴露

## 背景状态
- **主角行为体**：王林已夺舍马良身体，凭马良记忆行动，对外仍是“马良”。真马良已死。
- **同行三人**：周紫虹、杨姓师兄（资格最老）、林涛（性急，率先冲出），皆为战神殿弟子，原马良亦属此列。
- **处境**：域外战场持续崩溃坍塌，更有诡异生物（游魂）吞噬修士魂魄、使其化为干尸。相较坍塌（人凭空消失），被游魂吞噬更为可怕。

## 传送阵机制
- 域外战场共有**四处大型传送阵**，平时关闭，仅在固定时间开启，此刻距开启还有两天。
- 进入域外战场者皆得一枚**玉符**：既可抵抗罡风，又是传送回归的信物；无对应信物则无法直接传回本国。
- 王林无赵国信物，占据马良身体后最佳去向是马良所在的**火焚国**——这也是他救下周紫虹三人的用意之一：便于抢一枚玉符。
- 传送阵外笼罩白色光幕护体，阵外漂浮大量欲进阵者化成的干尸，游魂在干尸间钻动却绝不触碰光幕。

## 关键情节
1. **宝物之失与残存**：装有葫芦及天逆珠子（石珠）等众多宝贝的储物袋，早被藤化元捏碎化为乌有；但王林仍能在神识中感到石珠，血炼之术的飞剑亦仿佛融进灵魂，只待出域外战场后寻地闭关探查。
2. **游魂惧王林**：一路飞行中，所有诡异生物对四人视若无睹，甚至有游魂扑向周紫虹却在触及瞬间惊恐逃离。三人由此断定异常源于王林，却因惧祸而不敢发问。
3. **必杀之人**：王林视许浩、葛阳为必杀之人——因二人知晓真马良已死，须除以绝后患。他借马良记忆联络三名游魂邻居代为搜寻。后邻居传识：二人已死，储物袋由一只双角游魂急速送来。
4. **异常游魂**：该双角游魂因吞噬众多灵魂而体质被彻底改变，王林从它身上感到与当初梦境空间中司徒南那元婴相似的“共性”。（王林累计吞噬游魂近万，视其为以神识状态存活、本能吞噬一切的生命体。）
5. **收奴收场**：王林命游魂击杀林涛。林涛乃第一个识破王林夺舍真相者，当即不作质问，自愿成奴并按额送出一滴魂魄精血，随即反手持剑指向杨、张二人以示效忠。杨姓师兄亦献精血归顺。周紫虹只问“夺舍时马良是死是活”，王林答“死”，她遂亦献出精血。三滴精血入王林神识化作三点微弱光点，一念可令三人魂飞魄散。

## 设定要点
- 王林不嗜杀，但夺舍之事绝不可外泄；出域外战场后，“五十年时间足以改变一个人的性格”即可掩饰。
- 游魂与元婴（司徒南）之“共性”为后续线索。

## 待续伏笔
- 出域外战场后择地闭关，探查融入神识的石珠与血炼飞剑。

# 《仙逆》第123-125章 剧情梳理

## 一、收奴与魂之真相（第123章）
- 周紫虹、杨雄、林涛被王林收魂魄精血为奴。周紫虹（出身战神殿，曾拒众同门追求、发誓不到结丹不选双修道侣）担忧侍寝之事，实则王林毫无兴趣；林涛怕被灭口，杨雄怕王林舍弃马良而夺舍自己身体。
- 王林向三个寂灭空间『邻居』索要一只游魂，得知魂之体系：
  - 番魂/吞魂无法直接吞噬他人魂魄，只能吞游魂壮大；游魂是低阶魂，既是武器、奴隶又是食物。
  - 魂无法离开此地。游魂一旦进入**生灵空间**即为『魔头』，数量少则浩劫，多则生灵空间沦为新的**寂灭空间**，其中必有一个终成吞魂。
  - **寂灭与生灵之间有界律法则**，妄图过界者一律抹杀。
- **域外战场**：生灵界力者在生灵与寂灭之间开辟的混乱之地；即便如此游魂仍不能钻出空间裂缝，唯此地将崩溃、寂灭之魂进入后它们才得入。邻居称最终仍要随其回归寂灭空间，这是吞魂的使命。

## 二、试探界律法则（第123-124章）
- 王林驱三奴进入传送阵后独坐观察，一路尸体上的游魂皆避让王林；一只**变异游魂**（具智慧、能传哀求之色，同类畏之）始终跟随。
- 王林逐次驱一只、十余只、上千、逾两千游魂自杀式冲击传送阵光幕，皆被瞬间出现的**黑丝**钻体消散。得出：
  - 传送阵光幕即生灵空间入口；黑丝从四周无规则出现，专为灭杀灵魂而生。
  - 变异游魂能硬抗黑丝短暂冲入光幕、化一修士为干尸，随即被十余黑丝灭杀。
- 王林以一丝神识试探，迎击黑丝，任其钻入本体后以庞大神识包围研究，确认黑丝对灵魂伤害极大、数量过多连他也难承受。

## 三、备阵（第124章）
- 许浩、葛阳杀马良后被大量游魂『特殊照顾』吞噬（游魂放弃他人专找二人），储物袋因此完整。王林得**三个储物袋**：马良传送玉符、约两千余块中品灵石、数百法宝、材料齐全（二人近五十年存货）。
- 王林凭空间裂缝内所学《阵法基础手册》，布置**龟甲玄灭阵**：无攻击力但防御佳、可随身移动的基础阵，极耗灵石。每块中品灵石画一符号，每套49符号+阵眼灵石按于眉心。共布二十一套护身阵。

## 四、极境神识·冲阵离场（第125章）
- 域外战场加剧坍塌，空间裂缝遍布。三奴先以玉符传走，王林算准传送需三秒危险期后跃入。
- **第一秒**：数十黑丝射来，一套套阵法（近千符号）接连破裂挡下。
- **第二秒**：数百黑丝齐射，剩余符号与数百未祭炼法宝尽碎，约半数黑丝被耗，余者钻入王林体内，神识破损但坚持不灭。王林狂笑『老子已经死过一次，谁也不能阻止我回去』。
- **第三秒**：由无数黑丝组成的界律法则**大手**探出抓来；域外战场彻底坍塌，仅余四座传送阵悬于虚无。王林神识在大手拍下时崩溃又被『极』的神秘力量重组，反复上万次不断提纯。
- **极境之源**：这是王林第四次触『极』（前三次：丛林废墟、决明谷、寂灭空间灵魂苏醒）。因灵魂融合天逆珠子本有微乎其微极韵，在界律强压推动下，神识渐缩、极境渐扩，史无前例地融合——极烙印于**神识**而非灵力。
- **极境规则**：极具单一性；极境烙于神识者灵力将失原威，但神识可轻易灭杀一切同级、无视初/中/后三期；**极境终点为元婴后期**，上古仅一疑似者突破，天地法则绝不容元婴期以上极境修士。
- 神识终崩溃、驳杂尽去，王林剩下**一丝接近本源的极识**；第三秒终了，传送阵开启。

## 关键设定沉淀
- 空间层次：生灵空间 ↔（界律法则/传送阵）↔ 域外战场 ↔（空间裂缝）↔ 寂灭空间。
- 界律法则：以无规则黑丝、终极化为大手，抹杀过界之魂。
- 王林是已知唯一在神识上落下极烙印者。
