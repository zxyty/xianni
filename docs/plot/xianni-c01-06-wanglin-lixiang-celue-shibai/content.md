## 概述
《仙逆》开篇第1-6章（离乡 / 仙人 / 测试 / 无情 / 回程 / 势利），讲述主角王林（小名铁柱）作为凡人少年首次踏入修真世界、参加恒岳派收徒测试惨败、归乡尝尽世态炎凉的前史，奠定全书冷硬黑暗、逆天因果的基调。

## 人物与背景（第一章 离乡）
- **王林**：本名王林，小名铁柱（幼时体弱、父怕养不活按习俗取贱名）。聪慧好读、被村中公认神童。测试时15岁，过年即16。
- **家世**：王氏家族祖上木匠出身，在县城为望族、有数家木器铺。父亲**王天水**为家族庶出次子，不得接管家业，成婚后离乡定居山村，靠精湛木匠手艺过着小康日子。
- **四叔**（王天水之弟，王家老四）：在家族中较有出息，多年帮衬王天水卖木雕。恒岳派今年收徒、家族有三个推荐名额，分到四叔一个；四叔因自家儿子（虎子）只擅武不擅读，且念及幼时受王林祖母（二娘）照顾之恩，将名额让给王林。
- 父亲郑重告诫王林永世不忘四叔恩惠。

## 乡里庆贺与族中势利伏笔（第一章后半）
- 消息传开，十里八乡村民、王氏族人陆续来贺；王天水大办数百桌宴席。
- **王天山**（王天水大哥）携子**王卓**到场。王卓相貌俊美、傲慢轻蔑，当众贬低王林‘没那块料’，自称被恒岳派**道虚上仙**看中、内定弟子。
- 揭示旧怨：当年父亲临终本留王天水一份家产，王天山费心联络亲戚抢走并将王天水逼出家族。

## 见仙人与抵达山门（第二章 仙人）
- 张姓白衣青年驾剑光而至，接走王家三名少年（王卓、王浩、王林）腾云而去。
- **恒岳派山门**：赵国为数不多的修真门派之一，山明水秀、世外桃源；主峰顶有七彩大殿，殿旁一座弯月状长条石桥延伸入虚空云雾、连接另一山峰。500年前曾统领整个赵国修真界、拥数位元婴期老怪，历经变故后已衰落。

## 三项收徒测试（第三章 测试 / 第四章 无情）
主持者为黑衣中年人（三师兄）。共48名被推荐少年。
1. **灵根（天资）测试**：手按头顶判合格与否。连续十余人不合格；王卓合格（道虚师叔提过），另有一少女合格。**王林不合格**。
2. **王浩插曲**：王浩资质差，但献上一只打不开的玉盒——内封印一株三百年灵芝；黑衣中年人大喜，收其为身边炼丹药童，可同门修炼仙术。
3. 灵根合格仅3人；其余39人进入**毅力测试**：限三日内爬上山顶石阶，半途可呼喊放弃由弟子救下。王林血肉模糊全凭意念用双手攀爬，一中年修士评‘稚子心坚，奈何大道无情’‘毅力绝佳，可惜资质太普通，无缘’。第三日日出王林距顶不到十丈，时限已到判失败昏迷。39人中放弃25人，坚持到最后11人。
4. **剑灵阁测试**（白衣青年主持）：能走进摆满古剑的房间、获剑灵认同者，前两测不合格也可破格收为真正弟子。王林是11人中表现最好者之一，独自越过五丈线、推进至距门三丈处终被无形巨力推退；另一少女入两丈。**11人无一合格**。

## 回乡与世态炎凉（第五章 回程 / 第六章 势利）
- 张姓青年送王卓、王浩、王林返乡。王卓得意宣称十年内必成恒岳派翘楚；王浩成药童；王林‘三项测试无一合格’。
- 王氏家族正大摆庆功宴，亲戚原本极力奉承王天水。
- 仙人离去前留言‘大道无情’‘修道者不能有俗世牵挂，三天后再来接你们’。
- 真相揭晓：王卓拜入道虚门下、王浩成药童、**唯王林落选**。
- 亲戚嘴脸瞬变：先前阿谀者纷纷冷嘲热讽、远离王林一家，有人当场索回贺礼，悔婚者亦有。王天水忍无可忍与王天山争执，四叔以江湖朋友相护出面震慑。
- 王林强忍心中撕裂剧痛，冷眼旁观，**牢牢记住每一张势利嘴脸**——为后文性情埋下伏笔。

## 关键设定要点
- 修真门派层级出现：记名弟子、真正弟子、药童；境界提及‘元婴期’。
- 凡人对‘仙人’充满膜拜与不可及之感；恒岳派为凡人可望不可及的存在。
- 主题基调：大道无情、人情势利、逆境磨砺。
