# 仙逆 第251-252章 归隐·法宝

## 一、第251章 归隐——抵京都、遇神秘老者

- **心境与修为**：王林随车队夜行，晨吸紫色气龙（凝气吐纳）淬炼，体外红雾一夜再凝缩十分之一。此为四百年来首次因心态平和而得的天地气息之感。此时王林修为为**元婴中期**。
- **告别车队**：车队抵四级修真国京都。王林与凡人**吕兴**、让马的**孙老二**、随行郎中老者、白纱女子及俏丽丫鬟告别。丫鬟代小姐赠**十两金子**作诊金——此为王林四百年后再次触及凡间钱财。吕兴、孙老二均许诺'受欺负来找我'，浑不知同行者远超其先祖所见之修士千百倍。
- **京都·仙木**：城中心九根黑色圆柱（最小需十人环抱、最大需百人环抱），即'**仙木**'，传为仙人庇护降仙谕、征全国工匠所建。柱内各有修士打坐（修为最高结丹后期、最低筑基后期），外有阻绝神识的阵法，对王林形同虚设，散发九道灵气漩涡。京都内可见十余名修士（最高凝气十二三层），为四级修真国特色。

### 神秘老者
- 一邋遢老者常被打（偷窥女眷换衣），却能**一眼看穿修士修为**：见筑基说结丹、见结丹说元婴、见王林（元婴）便说化神，以'无人指点此生无法晋阶'为饵，专骗修士请客吃喝、逛青楼，被骗者反觉遇高人而恭敬。
- 老者以凡人之相示人，却看穿王林修为，王林判断其修为**远超化神期**故隐匿不可测。
- **点化**：老者称化神须**感悟天道**、产生**意境**。青楼中老者暗认出王林为'元婴中期'，一度考虑是否相帮，旋即被美色抛诸脑后。
- 王林感其神秘，决定少惹，收回神识。

### 归隐决定
- 悟'**若要化神，先要化凡**''小隐于野，大隐于市'，王林以**八两金子**在京都城西偏僻处租一小店，隐于繁华凡尘，感受轮回与天道，为突破元婴入化神作全面准备。

## 二、第252章 法宝——木雕寄情

- 王林如凡人般打理店铺，闪身外出，储物袋内存回连根拔起的大树，去皮削成木方。
- 忆童年其父手把手教做木雕，右手运刻刀，体内灵力顺刀融入木中。首成一慈和中年男子木雕（其**父**，粗布衣、手有粗痕），散溢灵力却令王林悲从中来（'爹，孩儿想你'）；次成慈爱**母**亲木雕。
- 一月间不辍雕刻、几近停修，店内摆满故乡山村居民男女老少木雕，以另类方式'复活'旧人。
- 一日刻一散发洪荒之气的**蛟龙**——即王林与**李慕婉**在**修魔海**时所遇的第一个灵兽。
- 铁具铺子一虎头虎脑少年（神似当年**王浩**）入店讨要木雕，王林顺手赠其**王浩**木雕；少年之父见木雕生异，欲付钱，王林言'只是一块木头'相赠。此时店内尚无一客光临。

## 三、世界状态归属提示
- **王浩**（第222章已死）、**李慕婉**（第188章已死）在本片段均为**回忆/木雕**形式出现，非新行动。
- 本片段王林以本名/'王老弟'凡人身份行事，无化名冒充情形。

## 关键伏笔
- 章题'法宝'与散发灵力的木雕，暗示这些木雕或为日后炼制法宝之引。
- '意境'成为王林突破元婴入化神的核心关窍。

# 第253-254章 蛟龙木雕

## 场景
王林隐居于一座远离修魔海的凡人城池（有京都、南王府、南城周木匠等），接手一间店铺开木雕铺。他辟谷已久却重拾家常饭菜与果子酒，四百年杀戮后首次融入凡人世界，体会凡人苦甜，心境平和，体内灵力以前所未有的速度流动。

## 木雕与灵威
- 王林所刻木雕栩栩如生，凡人只见精致，实则每件都凝聚庞大灵力，被限制在木雕之内无法扩散。
- 灵威堪比低阶法宝，部分甚至比肩中阶法宝——"以朽木制作的修真法宝"。
- **修为越高者感受越强**：筑基中期青衫修士触碰蛟龙木雕几乎筑基崩溃；结丹黑衣修士险些金丹不稳；凡人则几乎无感。
- 王林一年来至多只能雕出**中品灵兽**（如貌似乌龟、力大无穷的"钟鼓"）。**上品灵兽相当于人类化神期**，无论如何雕不成，最后几笔必失控碎裂。
- 半成品箱中放着孟驼子、（吉帝/吉神）、六欲魔君、上品灵兽、荒兽等木雕，共同点是**全部化神期以上**，无法刻成，强刻则化为飞灰。
- 王林信念：**当能刻下化神期修为之人或兽时，修为定会突破**。父母亲人的雕刻之外，他视这些木雕仅为平和心态、感悟天道的工具。

## 红雾煞气
王林闭目时体外浮现一掌宽红雾煞气，一年间凝炼压缩速度倍增。他并非强制压缩，而是修为达到一定程度后对未来产生气机预感，遂自然而然开始凝炼，认为放任其消散日后必悔。

## 人物：大牛
凡人少年，十二岁，铁匠夫妻之子，唤王林为"王叔/王叔叔"。一年间几乎每日携父亲自酿果子酒来看王林雕刻，痴迷木雕，想弃打铁学此手艺。见蛟龙木雕卖出十两金子（其父打铁一月仅得一两且属难得），下定决心愿看六十年学艺。

## 蛟龙木雕的流转
1. 王林年前雕成一条盘身望天、鳞片清晰的蛟龙木雕。
2. 徐涛（徐家店铺大儿子，现为南王世子亲信随从）经大牛引荐入店，一眼看出木雕胜过京城名匠周木匠——差在一个"生"字（栩栩如真 vs 栩栩如生）。以**十两金子**购得，以外套包裹匆匆离去。
3. 徐涛回南王府献给**南王世子**（紫袍青年，凝气期八层修士，孝子），世子大喜记其大功，并以魅姬相赏。
4. 世子携木雕至正殿献给父亲**南王**（俊朗威严的中年王爷，唯一爱好即雕刻品）。

## 南王府的两位修士
殿中两位道袍修士：
- **黑衣修士**（师兄，结丹/金丹期）：始终闭目，鉴出木雕所刻与门派典籍中蛟龙"几乎一模一样"，虽"没见过此兽"却"认识"，握雕时金丹险些不稳。
- **青衫修士**（师弟，筑基中期）：抓起木雕被沧桑之气冲得筑基欲崩，大骇松手；被师兄点破"始终不能突破"的缘由。

## 要点
- 此地距修魔海极远，妖兽种类繁多，即便修真者也难尽识王林所刻之兽。
- 果子酒甘甜带辛辣，下腹火热，成为王林每日必需，助其灵力运转。
