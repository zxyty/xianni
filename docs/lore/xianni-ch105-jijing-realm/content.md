# 第105章 极境临世

## 一、极境·灵力境界设定(核心lore)
上古时期存在三大难以琢磨的**灵力境界**:**极、道、始**。
- **神秘**:道境 > 极境;**莫测诡异**:始境 > 极境;**可怕**:极境 >> 道、始两境。
- **极境特性**:令灵力属性彻底偏向极端,拥此境者冷酷、无情远超旁人数倍;可将任何法术发挥到难以想象的极端。**同等级修士,极境无敌**。
- **明显缺陷**:无法达到化神期,**元婴后期即为极境修为终点**。但元婴后期的极境修士,普通化神期高手也不愿招惹。
- 若把魔道比作洪水猛兽,上古时期魔道与极境修士相比「乖巧得不得了,根本不是一个层次」;极境在上古被喻为真正的**逆天**。

## 二、极境的可怕先例(闪回/传说)
- 上古某位**超越婴变期**的逆天强者,度劫时被发现怀有极境灵力——震动恐惧整个上古修真界。此人横行修真界三千余年,为未冕之王,稍有不和便灭对方满门;后人推测**上古修真界的覆灭与他有莫大关系**(那三千年门派被屠杀一净,浩劫来时毫无抵抗力)。
- 其后几代极境传人皆被卡在元婴后期,化神限制仅在传说那人身上破例过一次。
- 记载多已散佚,仅高等修真国某些超级大派藏经之地留有零星描述:
  - **古涅国灵巧宗**天才:十年入结丹,门派被灭五年后横空出世,一月连杀近万人,以元婴初期力抗化神期,终由正魔两道请出婴变期强者才诛杀。
  - **清水国储君**:拜入某门派,三十年后出世,七日灭尽清水国所有修真者(含自身门派);被围杀时以血海沸腾、以血杀敌,留「浩浩之名」后神秘失踪(疑已度劫)。
- 共同点:皆为极境者。得境之因不明,或说遭逢巨变,或说共有同一件传承宝物。

## 三、王林入极境(本章剧情)
- 无人知晓:此刻三级修真国**赵国**的**决明谷**内,青年**王林**(修为筑基初期)正慢慢蜕变、迈向真正的极境。**藤化元**正是促成极境传人的催化者(其本人若知恐怕绝不会为)。
- 王林半跪于地,身外方圆几十丈被蓝色冰晶覆盖成绝域;胸口**天逆珠子**前所未有闪烁黑色光芒,飘起悬于头顶。
- **藤化元的威胁**:利用王林至亲血脉相惜,在其脑海留话——「我在决明谷外等你开谷之日,若你不出现,我就毁掉魂旗,让你亲人魂飞魄散。」

### 天逆珠子与天道门
- **天道门**二男一女(含一**筑基中期**修士)追至,被冰晶气息所慑。女子认出珠子眼熟;另一男子失声:天道门**始祖**玉简所记珠子刻的是**云朵**,而这珠子上却是**树叶**——图案不同,但大小外表与玉简描述一模一样。
- 始祖悬赏:上缴珠子者获赠婴变级法宝一件,所属门派获赠十具化神期炼尸,所在修真国级别提升一级。三人贪功,决定不通知他人、私自夺取。

### 极境屠戮(第一大功效展开)
- 王林睁眼,眼中血色尽失、阴沉似水、充满死寂与杀机。**极境第一大功效——同境界无敌**就此展开。
- 一挥手,冰晶飘起瞬间包裹天道门三人,三人动作定格,随即冰晶碎裂、倒地身亡。王林收回天逆珠子珍重怀藏,穿身而过。
- 一青年欲杀人夺宝,飞剑刺来即被冻成冰坨;**绿色小剑(古怪飞剑)**一闪,自其胸口穿过,身化冰晶。
- 一天后谷内正魔两派斗法处,王林旁若无人走过。魔道中年男子掷雷符,蓝色冰晶竟顺雷电冻住无形天雷;该男子随即被绿光穿胸化为冰晶。挡路四人亦尽数成冰。
- 正道**元天派(元兲派)弟子周权**抱拳询问门派,王林置若罔闻。一魔道年轻弟子暗生杀气欲报师兄之仇,被王林回眸死寂目光所慑;王林转身后,该弟子胸口绿光一闪,喷血倒地化为冰雕,连喷出的血也在半空成冰。众人噤若寒蝉,待其消失后各自撤退。

## 四、要点
- 极境=灵力境界层面的属性(极/道/始),独立于凡人→化神的境界阶序;可与筑基等修为并存(王林筑基初期已显极境威能)。
- **天逆珠子**(树叶图案)与天道门始祖所求的**云朵图案珠子**为不同来源的同类宝物,疑与「拥有同一传承宝物即得极境」的猜测相关。
- 藤化元以魂旗(囚王林亲人魂魄)要挟王林开谷相见,埋下后续因果。

# 修真国等级体系与域外战场、决明谷设定（第106-107章）

## 一、修真国等级晋升机制（第106章）
五级以下修真国，除特殊事件外，提升等级只有一种方式——本土修士独自修炼至更高境界：
- 一级修真国出现**结丹期** → 自动升为二级；
- 出现**元婴期** → 升三级；
- 出现**化神期** → 升四级；
- 出现**婴变期** → 升五级。

此类升级除需时间外，更需莫大机缘，越往上越难，往往千年一遇，三升四、四升五更是万年未必成功。达成者无一例外成为该修真国领袖。

## 二、等级差距与域外战场资格
- 四级修真国方有资格参加域外战场，但实力远不能与五级媲美。**境界差距非数量可弥补**：四级顶尖的化神后期，在五级中沦为二流；一个婴变期高手即可灭一个四级修真国（史上多次发生）。
- 故四级的域外之战，实为各自靠山（五级修真国）之间的对抗棋子；真正决定性的域外战，在五级修真国之间展开。

## 三、五级的困境与升六级
- 修真国一旦达五级，最严重问题是**资源与材料**：庞大国力与修士消耗巨大，靠抢劫低级国也无法补充，唯一出路是升六级。
- 升六级已非个人修为可决，须经**域外战场**。域外战场是一片充满裂缝的无边空间，遍布可轻易灭化神期的**罡风**与随时出现的**空间裂痕**；非婴变期被罡风吹到立刻熔为灰，空间裂痕连婴变期老怪也难逃，唯六级修真国强者可来去自如。
- 每隔几百年有一场域外大战，交战方均为四级以上修真国，由修真联盟推动。**连续获胜超千次**者升为六级，获大量材料功法，并可选送一代表受联盟传功直接超越婴变期；最关键奖励是**获赐一颗星球**作为基地。

## 四、六级修真国的考核
第一步：域外战场连续胜千次、获赠星球。第二步：规定时间内所属修真星内须有数个完整的一至五级修真国以形成循环。同一修真星上的六级国一般不干涉五级之战，但若打压则该星难出第二个六级国；推测六升七或与所在修真星曾出过几个六级国有关。

## 五、域外战场清理与决明谷
- 战场遗留大量尸体与废弃材料需定时清理，六级修真国规定：**清理只准三级以下修真国**处理，且进入者修为不能超过结丹期。这些材料对低级国不亚于天才地宝。战场依四级国背后的五级派系划分为数个区域。
- 即便三级修真国**赵国**亦为争夺清理工作而有**决明谷**之存在。
- 决明谷外有巨大封印阵法，为数千年前一四级修真国的**化神期高手**奉命布下，凡结丹期以上、化神期以下者均无法进入——此即藤化元虽经天算之法得知王林在谷内，却无法进入杀他的原因。
- 决明谷亦是域外战场设在赵国的**唯一入口**。资格确定后，留谷的筑基期修士即被传送到战场，直到五十年期满才统一传回。藤化元正因知此，才屠王林全族收魂，逼王林出谷；否则想杀王林只能等五十年后。

## 六、王林线剧情
- 三日间死于王林手中的修士不计其数，凡冒犯者一挥手化为冰晶，筑基后期亦难抵挡，谷内众人对其闻风丧胆；有人分析他已达**结丹期**（谷内推测），却不知他如何进入谷内。
- 三个月里，正魔双方多次交战，但见到王林即停手让路。王林一路向北，杀机反而更浓，脑中空白。
- 数日后至决明谷最北陡崖底（爬上即出谷，但封印阵会将离者传回）。王林思念父母便心口绞痛、喷血、身子渐虚，体内灵气却越发迈向极境。跪地磕头、流下血泪：“爹，娘，铁柱不孝”。
- **司徒南苏醒**（五天前醒），告知：藤化元屠其亲人是为逼他现身；**魂旗**是元婴期方能制作、可封印魂魄的法宝，只要魂魄不散，待其恢复修为自有办法复活亲人。司徒南所剩元婴精华不多，只能支持**三次瞬移**，须谨慎使用；并建议王林前往域外战场，可缩短其恢复时间，随后再度沉睡。

## 七、第107章 米粒之光——灵力异变
- 王林祭亲后神智恢复，挖穴静坐参悟。此段时日灵力冰冻威力远胜从前，连司徒南也无解。由此可见**极、道、始三境已在六级修真国销声匿迹**。
- 内视发现灵力已变深蓝色，由无数深蓝细丝组成——细丝即灵力变异之因（某种神奇物质含量骤增）。
- 王林凝神压缩拳头大的蓝色冰球，缩小三分之一时细线大量崩溃，反复补充灵力炼化，数个时辰后终于完成：冰球缩至**米粒大小**、颜色由深蓝变为**青色**，内里被细丝完全占据，散发危险气息。王林抓米粒飞出洞穴，掷向密林大树。

## 一、青色米粒的炼制与威力
- 王林以自身灵力压缩、专门提炼其内细丝，制成不起眼的「青色米粒」，自称「这青色米粒，就是最好的法宝了」。
- **威力**：引爆后化为一圈深蓝色环形波纹，如怒涛巨浪向四周推开三百余丈；方圆三百丈内树木花草昆虫乃至地底小虫等一切生命尽数冻毙，成为生命禁区。连王林自身被蓝色光环扫及也一度身子僵硬，靠体内同源灵力融合方恢复。
- **限制**：制作极耗灵力，做一个已吃力，三天仅得三个；炼第四个时必致原有三个消散其一，故三个为极限。
- **融合实验**：两个米粒可无声融合（颜色略深），威力可增且不占名额；三个融合则立即自爆（融合第三个时险些被波及，靠一次瞬移逃生，波纹扩至千丈）。最终确认「三个是极限，融合成一则只算一个」，得以在融合基础上再补炼。整理米粒耗时一个多月，此时距决明谷封印开启已不多。

## 二、绿色小剑（血炼飞剑）
- 王林血炼之物；因体内灵力异变，飞剑间接带上「极」（冰）属性，每次闪烁便使一名修士化为冰雕暴毙。
- 异变后飞剑控制不似以往得心应手，飞行间偶有一丝滞留，王林需强行加大控制力度压制。
- 王林自觉灵力异变后心中常存一股难以自控的杀意，索性不再压制。

## 三、决明谷·资格令牌争夺赛
- **规则**：门派间抢夺「资格令牌」，共七块。域外战场通道开启时，若剩余超过三块令牌未被销毁，则取消赵国进入域外战场的资格；进入名额仅三个。（王林入谷前由夜自在告知，并结合此前司徒南苏醒时的简单介绍推知——按世界状态，司徒南死于第61章，此处为回忆引述。）
- **李山盗令**：玄道宗李山（凝气期，擅造假法宝药丸与偷技）获朴南子破例带入，凭一件可掩盖痕迹的法宝多次偷取，共窃得四块令牌；偷第五块时法宝失效被发现，玄道宗集结地遭群起围攻，李山从地道逃出后被上百筑基期修士追杀。
- **王林现身**：以引力术甩开阻拦者，绿色小剑连斩围拢的筑基期修士，无人能挡（极境灵力下敌人法宝尽成冰坨）。逼李山交出令牌，取走地上三块。

## 四、冻杀无锋谷与玄道宗
- **无锋谷**：假丹境界老者（潜修数年或可结丹）持无锋谷重宝「天雷风火伞」（三寸小伞、金色火焰怪风），轻视王林出手。王林以引力术控青色米粒撞入，引爆，蓝色波纹将怪风化为蓝晶、金火冻结，老者及无锋谷十七名弟子（含逃至十余丈外者）尽成冰雕，天雷风火伞被蓝晶覆盖。
- **合欢宗**：领头女弟子及时撤退幸免，献出令牌并交代关键情报：令牌共七块、名额三个、剩余下落，并指认玄道宗集结地尚有令牌。
- **玄道宗集结地**：王林令李山召出被困的玄道宗十人。马姓老者（马师兄）出言不逊放出飓风，被王林一剑冻杀，气绝身亡——柳风震惊于「以马师兄筑基中期修为亦一剑之下身亡」。

## 五、故人：柳眉与柳风
- **柳眉**：昔日恒岳派相关的娇媚女子，现为玄道宗弟子，已由始祖朴南子亲自施法助其筑基。自始至终一言不发，只深深注视王林，干脆交出令牌；王林收令后转身离去，柳眉幽幽一叹。
- **柳风**：柳眉之兄，年约三旬，玄道宗，筑基初期（灵基不稳，刚晋级不久），认出王林为「恒岳派」旧人。
- 王林由恒岳派往事、爹娘容貌浮现心底而心中绞痛。

## 六、结局
- 王林先后取得共五块令牌，回洞府后捏碎四块，只留一块揣入储物袋，用引力术封死洞口并布预警阵法，开始闭关，为两个月后的大战做准备。
