# 黄泉升窍决与极阴之地体系（第87章 黄泉寒丹）

## 极阴之地体系（司徒南讲解）
极阴之地品质越高，对修炼者裨益越大。按本质分四种：
- **黄阴**：最简单，死亡大量生灵之地即成，死人越多越佳。
- **玄阴**：较难，多出现在极冷之地。
- **地阴**：更难，需莫大机缘，乃地底深渊散发的气息。
- **天阴**：几乎不可能，须地、玄、黄三者同时具备，方产生奇妙转变化为天阴之地。

品质高低排序：黄 < 玄 < 地 < 天。

### 品质测定法术
王林掐诀念咒，凝出红色光球（专测极阴之地品质）。光芒四色对应四类：
- 紫→天阴、蓝→地阴、银→玄阴、红→黄阴。

### 层次与品阶
每种极阴之地再分四层次：**普通 < 上佳 < 沉阴 < 绝阴**，每层次又分十品。光球闪烁次数表品阶（如闪五下=普通五品；红球一分为二再闪=上佳一品）。
- 普通三品以上：可修黄泉升窍决。
- 普通八品以上：可加快修炼速度。

## 黄泉升窍决
司徒南所知法诀中目前最适合王林、且较快速的一门。
- **结构**：分九层，每修三层在体内凝结一枚寒丹；九层全修完，三颗寒丹相互融合，**有一定几率直接攀升至结丹期**——此为王林决心修炼的最大原因。
- **冲窍**：靠冲窍凝丹。三大穴窍=**丹田、气海、祖窍**。三窍冲破瞬间凝出寒丹，寒丹一现，体内灵力立刻转变为阴寒之力。
- **冲窍成功率**与极阴之地品质有莫大关系，故须循循渐进、选适配自身的品阶之地（司徒南沉睡前再三叮嘱）。
- **更快的替代**：夺丹（活捉结丹期高手），对当前王林近乎自杀，故不取。

## 王林的修炼地选址
- 一处人口不下千万、瞬间化为死寂废墟的黄阴之地。
- 测得：起始普通五品；深入数百丈升至普通八品；至石珠所在残骸前达上佳一品（阴寒之气如水中漫步、窒息感、血管刺痛、灵力将失控）。
- 王林退回**普通五品**位置——他分析后认为最适合自身身体、契合黄泉升窍决循序渐进之需。
- 白天阴寒之气随日出全消；废墟极大，当前位置不及万分之一，越深阴气越浓。

## 阵法布置
- 依两年前在藤家城用中品灵石换得的《阵法详解》（含数个阵图、操作摆设介绍）。
- 阵法博大精深，衍化之术讲究推算之力，越往后越难；王林仅勉强了解一二。
- 照葫芦画瓢，用**引力术**抓石块摆出「**小迷踪**」阵（粗糙、无镇旗），仅能困凡人，对修士形同虚设。
- 对策：多摆几个阵图叠加凑效，遂连续摆出数阵。

## 首次外界吐纳阴寒之气
- 这是王林**夺基后第一次外界吐纳**（此前两年在地底只吸收灵气液体内部消化）。
- 吐纳法：**一长短五**——吸气长，呼气仅为吸气的五分之一，以最快速度纳阴寒之气入体。
- 异变：以往吐纳灵气身外似有薄膜阻碍、入体慢且大半流失；此次薄膜如消失，阴寒之气铺天盖地涌入且吸收一大半。
- 三种自我推测（未深究）：①阴寒之气比灵气更适合他吐纳；②筑基期后吐纳本就快于凝气期；③吸收藤厉部分灵根后天资提高。
- 体感：阴寒属性冰冷，钻入五脏六腑，手脚冰凉、血流粘稠、心跳变缓，眉发汗毛结霜，一股阴寒之气透体而出与四周融合。

> 注：司徒南已于第61章死亡，本章相关讲解均为其生前传授内容之回忆。

## 第八十八章 蓝肤怪人

### 黄泉升窍决·阴寒之气修炼
- 王林腹部凝出无声转动的细小气团，缓缓散出阴寒之气；体内灵力与其互不排斥、各不相干。
- 顿悟功法命名：所谓‘黄泉’，即修炼时须将身体改为**濒死状态**，以即将步入黄泉之形态更适合吸收阴寒之气，稍有不慎即真身入黄泉。
- **诡功**：此功在六级修真国都少有人敢练，**司徒南**是少有修成者之一；魔道称之为‘诡功’，因变化诡异，或裨益极大、或功散人亡。

### 石珠新认识·阴寒露水
- 重大发现：神秘**石珠**在此地竟也能凝结露水，且其中蕴含的并非灵气而是**阴寒之气**，对石珠又有新认识。
- 王林开始积累阴寒露水，**子夜**阴寒之气最浓时露水最易出现。
- 服下后判断：阴寒露水之效仅相当于灵气露水中‘**雪水**’档次，远不及灵气露水。
- 修炼安排：白天入**梦境空间**以阴寒露水修炼，夜晚在房间吐纳；功法本质=吸收阴寒之气越多威力越大。
- 吐纳渐与四周阴寒之气融为一体，心跳渐缓近乎若有若无，几次几乎停跳仍坚持下来。月余后体内阴寒之气渐多，自评‘应可第一次**冲窍**’。

### 异常预警
- 清晨发现昨夜失血而亡的几只野兽尸体被拖入废墟深处，阵法无被攻迹象，自己毫无察觉，警觉废墟内或有他者。
- 王林佯装无事，重补两具兽尸为饵，加固阵法迷雾（风雷之声），夜里收敛吐纳、神识最大范围（百丈）散开，专盯兽尸位置；四周异常寂静，连虫蝇野兽嘶鸣都无。

### 夜战蓝肤怪人
- 子时阴寒气息如沸，废墟方向窜出**蓝肤怪人**：全身蓝色皮肤、每寸皮肤布满诡异符号、身上数处贴有**九张半寸黄纸**，速度极快，先掠走兽尸。
- 怪人绕阵迷雾试探后冲入，王林掐诀打白光、张口吐绿光反击，巨大推力炸开迷雾。
- 飞剑（绿色小剑）凭空穿透怪人心脏，蓝血出体即化冰晶落地——怪人受穿心仍能行动。
- 王林祭出古朴**剑鞘**，飞剑入鞘下沉至**五分之二**，颜色由绿→蓝→黑，离鞘斩中怪人胸口将其抛飞；飞剑被蓝血粘住、难以驱动、瞬移失效。
- 王林取出**得自张虎的纸符**追击：怪人以诡异角度扭身落地、胸口碗大伤口，符号飞闪快速愈合；纸符自燃化乌光（浓郁热浪）再中其胸，怪人被穿透却仍不死，飞剑借此脱身归来。
- 怪人撕下九张黄纸之一，青色气体疯狂涌出包裹全身护体。
- 王林飞剑归鞘下沉至**五分之三**（自身当前极限），剑色由黑变红；自评筑基初期用此剑对筑基中期亦有一搏之力。

### 状态要点
- 王林：筑基初期；练黄泉升窍决；手握石珠、飞剑、古朴剑鞘、张虎遗留纸符等。
- 战斗未完，怪人青气护体、王林剑鞘下沉至极限红剑待发。

## 章节
第八十九章《唬人法宝》、第九十章《初露迹象》。地点：丛林深处的废墟城市（极阴之地，源自塔形建筑物的治疗光柱处可测得上佳一品）。

## 一、与蓝皮肤怪人的对峙与停战
- 怪人身负**九张黄纸**，撕下一张即可在濒死状态痊愈，故战斗难分胜负；其全身布满符号、力大无穷（一拳碎断壁），但极忌惮王林的飞剑/剑鞘。
- 双方**语言不通**。王林在断壁上画废墟草图，一分为二，划定左右领地并暗示杀机，以图示达成停战。
- 怪人**夺走小迷踪阵法**（举之如小山离去），随后去而复返，以示歉意。

## 二、幻术圆珠（唬人法宝）
- 怪人赠王林一枚**拳头十分之一大小、沾尘带裂痕的圆珠**，散出微弱灵力波动。
- 注入灵力即冒白烟幻化出一名鹤发鹰目老者，**注入灵力越多，幻化老者表现的修为越高**：初为凝气期十五层 → 筑基中期 → 直至看不出修为、气势巅峰，傲然道"结丹期以下，没资格与老夫一战，滚"。
- 本质是**幻术法宝**，逼真度近乎实质，专用于唬人。每次催动后圆珠裂痕增加一道。
- 王林由此推断：怪人不会用法宝，且废墟城市内必藏更多宝物——废墟在其眼中成了神秘宝藏（已知有治疗光线、幻术法宝、神秘怪人）。

## 三、黄泉升窍决·首次冲窍（核心剧情）
- **功法体系**：诡功，讲究"不破不立、先破后立"。三大窍穴＝小腹丹田、胸口气海、脑部祖窍；每窍须碎立三次循环方成寒丹。司徒南曾重点指出第一、四、七层（即首冲三窍）极难炼。
- 王林两月吐纳积成浓郁阴寒气团后开始冲窍，过程剧痛、腹部转青起霜。
- **失控吞噬**：丹田现黑洞，先疯狂"吸收"、二次开窍后变为"吞噬"，方圆百丈→千丈阴寒之气如涨潮齐聚一点，形成真空带。司徒南正沉睡，王林独力应对（黄泉升窍决无此变记载，此即"诡功"得名之由，或大益或功散人亡）。
- 一鼓作气连成**第一层（丹田初开）、第二层（碎而复立）、第三层（凝出青色寒丹，触手扩散全身）**。
- 趁势强冲**第四层·气海穴**，连冲百余次失败，阴寒之气耗尽、天明无补，残余被寒丹吸收。
- **结果**：王林灵力几乎尽数转为**阴寒灵力**，量增逾一倍，达**筑基初期顶峰**。他领悟此"变态修炼"思路：威力大但易被撑爆，需有计划逐步操作降低风险；并决意搬至塔形建筑物附近（极阴上佳一品）以提升冲窍成功率。

## 四、章末伏笔：藤家寻仇与棺中存在
- **藤化元（腾化元）**：藤家老祖，全身散发毁天灭地气息的老者，为孙辈**藤厉**（已于第67章死亡）报仇而来。他循"诅咒"之感一路追来，被迷雾阻隔无法确定方位，只知仇人在此丛林内。藤化元忌惮丛林内一强大存在，本不愿入，终为复仇咬牙闯入（自忖不动一草一木只寻仇人）。
- **棺中存在**：废墟最深处一口棺材掀起一角，伸出紫黑枯手凝出紫色雷球，自语"**百年期限又临，我的主人，上一次你抛掉肉身躲过，这次你跑不掉了**"——身份成谜的强大存在。
