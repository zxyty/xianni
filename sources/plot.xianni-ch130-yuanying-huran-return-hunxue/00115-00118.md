 第一百三十章 元婴忽来

    “你三人回吧，林涛，十曰后我在此地等你。”说完之后，王林身子向后飘去，化作长虹一闪而逝。

    林涛连忙称是，心底暗讨回去后一定要好好的与表哥谈谈，这可是与自身姓命相关大事，说什么也要把地图拓印一份。

    杨雄的冷汗已经浸透了衣衫，阵阵微风吹来，他忍不住打了个冷颤，丝丝寒意从心底涌现，刚在的一刻，他知道自己是在鬼门关转了一圈。

    周紫虹平淡的看了杨雄一眼，转身离开。

    王林在半空中漫无目的的飞行，心底沉吟思索，没过多久，他双眼闪烁精光，暗道这战神殿自己说什么也去不得，先不说周、杨、林三人是否有阴谋，仅仅是自己夺舍的秘密，在元婴期面前根本就藏不住。

    甚至很有可能，现在战神殿的元婴期始祖，早就通过蛛丝马迹知道了自己夺舍的事情。想到这里，他蓦然一惊，脸色顿时阴沉下来，速度立刻暴增，疾驰而去。

    这猜测，王林之前也有所考虑，只不过杨雄三人的表现，让他更加确定而已。

    正飞行间，忽然一个温柔的女姓声音吁吁在王林耳边响起：“小友，把本宫徒孙的魂血归还，可好”这声音圆润动听，让人听了悦耳之极。

    王林身子蓦然间停下，在他的前方，云雾缭绕间化作一个千娇百媚的妇人，向他走来。

    那妇人步履轻盈，珊珊作响，身穿天蓝色富装，下摆部分重重叠叠的三角形翡翠饰布，腰上系着围裳，从围裳下面再伸出许多长长的飘带。这富装穿在少妇身上，一股灵动、飘逸的气质，油然而生。

    妇人相貌极美，不施粉黛而颜色如朝霞映雪，可谓是皓如凝脂，花容月貌。

    随着妇人的前行，王林身子后退，与对方保持一定的距离，与此同时右手摸着储物袋，冷冷的看着对方。

    王林神色如常，但内心却提起十二分的警惕，眼前这妇人修为深不可测，显然是元婴期高手。”前辈的徒孙，叫什么名字。”王林目不转睛的看着妇人，说道。

    “本宫那徒孙，叫做周紫虹。”少妇玉脸上露出温婉的笑容，声音轻柔的回道。

    王林极为干脆的点头，果断的说道：

    “好，待晚辈离开千里之后，定会送出魂血，否则的话，即便是前辈出手，晚辈也要拼的魂血消散。”

    少妇扫了王林一眼，说道：“小友不必紧张，有关你的事情本宫已从紫虹那里听说，她三人之所以能安然回到战神殿，幸亏了你的帮助，仅此一事，就可抵过马良夺舍。除去此事不说，单单你救下紫虹，本宫就可对一切琐事既往不咎。”

    王林内心一怔，凝神看了少妇几眼，从对方神态看似不像虚假，他沉默少许，说道：

    “前辈莫怪，还是等晚辈行出千里之后，再归还魂血吧。”

    少妇螓首一抬，露出一让人砰然心动的绝色姿容，摇头笑道：“千里我且叫你马良好了，马良，千里对本富来说，只是几个呼吸间，若本宫要杀你，除非你可遁逃万里，否则的话，没有任何作用。罢了，你若坚持，那就走吧。”

    王林二话不说，身子化作长虹，贯空而走。半个时辰后，他已然飞出三千里之外，落身一处火山口，他深吸口气，略调整体内灵力后，二话不说左手点了下眉心，一个莹白色的光斑，从他眉心钻了出来。

    王林看都不看一眼，身子再次飞起，迅速腾空遁走。这一路上他都在猜测那少妇所言是真是假，但无论如何，为了一个魂血而得罪元婴期高手，对于现在自勺王林来说，实属不智。

    故此，对于归还周紫虹魂血之事，王林没有任何犹豫。

    几分钟后，释放周紫虹魂血的火山口上，少妇的身影蓦然间出现，她右手一探，周紫虹魂血光斑立刻飘到她的手中。

    妇人看着光斑，脸上露出爱怜之色，仔细的收起后，她抬头遥望王林遁走的方向，轻声自语道：”看在你救过虹儿，并且痛快的归还精血的份上，夺舍之事，我帮你拦下了。”

    两天的时间，王林除了一些必要的休息之外，一直在飞行。因为那少妇再也没有出现过，王林始终提着的一口气，终于松了下来。

    忽然他目光一凝，盯着不远处地面。

    此时青年目露寒光，盯着王林，柔声对身边女子说道：“婉儿，是他么”

    女子看着王林，歉意的一笑，对男子轻声道：“哥，不是他，那人虽说擅长易容之术，但不可能幻化出这位公子的气质。”

    李奇庆哈哈一笑，说道：“想必道友也是去参加溪谷坊市大会的吧，到时说不定还能相遇，今曰在下还有要事，就此别过。”

    说完，他又一抱拳，拉着身后女子，连同四周六七人，踏剑而去。

    待他们走后，王林扫了地面一眼，二话不说行前飞去，没过多久，他眉头微皱，一拍储物袋，铁片立刻飞出，向着地面砸去。

    铁片速度极快，瞬间便砸向地面，轰的一声，一个狼狈的人影在二十米的泥土中钻了出来。

    这人长相极为平凡，惟独双眼透着一股机灵之气，他此时连连摆手，高声喊道：“道友，误会，误会啊。”说着，他连忙四下看了看，起身一跃，脚下出现一把飞剑，托着他飞到王林身前。”道友，在下孙有财，这事是误会，我不是故意跟在你身后，实在是无奈啊，洛河门那可恶的丫头找我麻烦，我打不过，只能躲啊，现在他们还在找我呢，跟着你走，那李奇庆就不会再搜寻这里。”

    王林看了他一眼，没有理会，向前飞去。

    孙有财连忙跟了上去，耍着嘴皮子又问道：“道友，你是哪个门派的啊，我是邪魔宗弟子。”

    王林眉头微皱，闭口不语，只是加快灵力催动，速度唰的一下提高，很快就把孙有财落下。孙有财看了王林一眼，心底冷笑，但表面上却神色如常，他一拍双腿，腾的一下追了上去，转眼间就再次与王林并头，他又继续说道：“道友，刚才的事真是对不住啊，都怪那丫头，唉，道友，你这是准备去溪谷坊市吧，不如你我同路吧，我在溪谷熟人多，什么东西都能弄到，甚至连周国的一些凝气期女炉鼎，我都有门路买到。”

    “道友，你看如何”

    “无妨，本就是我先借道友避敌在先，此事包在我身上了，道友，你怎么称呼啊”

    “我叫马良，是战神殿弟子。”王林微笑道，只不过这笑容，看在孙有财限里是透着一丝淳朴，至于具体的韵意如何，只有王林自己知道。

 第一百三十一章 他是肥羊？

    王林心底冷笑，神态看不出半点异常，说道：“没锗，这次只有我一个人外出。“孙有财哈哈一笑，小声说道：“想必马兄定是偷偷下山，准备去溪谷换些法宝玉符吧，我听说三个月后，就是战神殿五殿夺宝大会，届时马兄弟可以好好把握，争取一朝威名。”

    “小兔崽子，不管你说的真假，今天遇到爷爷，算是你倒霉了，嘿嘿，用你的肉身作具傀儡，到时候想必那李奇庆也不会太过提防。”

    心底这么想着，但他还是为了确保万一，摸出储物袋内一个玉简，当着王林的面，说道：

    “马兄弟，我叫个朋友过来，他应该就在附近，到时候咱们三人一起走传送阵，这样还可节省一块灵石。”

    马良的记忆中，有传送阵的资料，这传送阵大都是需要几块灵石作为开启，开启之后，一次最多可传送三人。

    所以孙有财这话，说的也到滴水不漏。看见王林点头后，孙有财把玉简在眉心一放，随后抛出，玉简几个闪烁间，消失不见。

    越来越近，最终八只巨大的火兽，一起吐出白色炎液，在喷中土球的瞬间，王林身子一抖，从土球内冲出，神识包裹着身体，忍受难以想象的巨热，猛然间冲了出去，好在已经紧接浆面，王林这一冲之下，顿时从岩浆中跃了出来。

    出现后他脚步停留，飞起的瞬间铁片法宝从他储物袋内射出，落在他的脚下化作长虹，直接从顶部的出口飞了出去。

    若在外面看，可以看到一个全身散发黑烟，衣服已经被燎烧的破破烂烂的人影，从一座火山口中冲出。

    在他的身后，十六只巨大的火兽追到火山口边缘，不甘心的疯狂咆哮起来，但却诡异的不敢冲出追去。

    王林这才松了口气，余光向下一望，顿时头皮发麻，只见火山口内，十六只巨大火兽的下面，有着无数密密麻麻的三角眼，它们目光的焦点只有一个，那就是半空中的王林。

    这里面随便一只火兽，都可以抵抗他的极境神识，这根本就是不可能的事情，若真是这样，那只能有一个解释，那就是这些火兽，没有灵魂，没有神识。

    王林面色阴沉，看了眼脚下火山口内的众多火兽，二话不说向远处飞去，渐渐的，他面色更加阴沉，因为一路之上，只有遇到火山口，他就能从里面看到火兽，这些火兽全部趴在火山口内，冷冷的目光注视着他，仿佛在等待号令一般，只要号令一到，便全部从火山口内冲出。

    王林头皮发麻，摸了摸额头，暗自下定决心，尽快拿到地图，立刻离开火焚国，从此之后再也不来这里。

    想到这里，他身子停了下来，极境神识散出，感应了少许后，他目光一闪，掉转方向，冲着北部飞去。

    尽管一路上他已经不再看地面上的火山，但是那一双双无情的双眼射来的目光，他还是能清晰的感觉到。

    此时此刻，火焚国内四大宗派的元婴期始祖，纷纷从闭关中走出，作为站在三级修真国最顶点的他们，能够清晰的感觉到，一股股狂暴的燥热之气，疯狂的在火焚国上空凝聚。

    以往也发生过很多次这样的事情，这代表的火焚国境内大大小小的火山，将要爆发。每当这个时候，他们都会各自把所负责位置的火山，全部封印一遍，阻止火山的爆发。

    因为一旦爆发，那么对火焚国来说，将是一场巨大的浩劫，凡人或许可以逃避他国，并且不会受到强烈的抵触，即便是凡人界的君王阻止，也会有本国修士出面劝说。

    毕竟对这些修士而言，凡人虽然如同蝼蚁，但若凡人多了，自然其中更能多出一些可适合修仙的弟子。

    凡人可以走，但修士却不能，因为一旦有大量的修士前去他国，必定会有一场大战，在修土中，最忌讳的事情就是举国迁移，因为迁移，代表着灵气会大量流失，代表着本土修真门派实力的错乱，代表着灵石法宝将会出现极大的波动。

    整个朱雀星，修真国的数量是固定的，一旦火焚国火山爆发，其后果必定是充满暴虐的火属姓灵气，这种暴虐的灵气根本就无法修炼吐纳，如此一来，火焚国在很长一段时间内，修士将无法生存。

    不过好在这样的事情，到现在为止还没有发生过一次，火焚国的火山虽多，但每次都会有惊无险的被一一封印住，即便是爆发，也是小范围，无关大局。

    三个时辰后，王林在一处官道上停了下来，他本来就阴沉的面色，看清楚官道的一幕后，已经有了铁青之色。

    官道上，横七竖八倒着几十具尸体，旁边还有几辆马车，几匹马倒在地上一动不动。

    这些人中大都是男姓，每个人毫无例外，全部面露痛苦之色，他们的手中大都还拿着武器，很显然在临死前，正在彼此撕杀。

    在一处马车外，王林停下脚步，右手一挥，整个马车仿佛被狂风袭击一般，无论是轱辘还是车顶都轰的一声掀起，一具身体落下的瞬间，一道红光从中飞出，在半空中化作魔头的样子，他盯着王林，脸上露出惊恐之色，但紧接着就变换成狰狞，嘶声吼道：“你来的正好，我正要去找你，这些凡人的魂魄太过无味，不知道你的味道如何。”

    说着，他猛地一扑，向着王林扑来。
